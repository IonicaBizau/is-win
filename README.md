> Check if the machine is running Windows or not.

## :cloud: Installation

```sh
$ npm i --save is-win
```


## :clipboard: Example



```js
const isWin = require("is-win");

console.log(isWin());
```

## :memo: Documentation


### `isWin()`
Check if the machine is running Windows or not.

#### Return
- **Boolean** `true`, if the platform is Windows, `false` otherwise.
